# Resumen Básico - React + CSS (Explicación Clara)

## 🎨 CSS - Palabras clave explicadas

### Selectores y Reglas principales:

- **`:root`** → Define variables CSS globales (colores, fuentes, etc.) que se pueden usar en toda la página.
- **`--bg: #f8f9fa;`** → Variable personalizada para el color de fondo. Se usa con `var(--bg)`.
- **`body`** → Estilos para el elemento `<body>` de la página.
- **`#root`** → Contenedor principal de React. Aquí centramos y limitamos el ancho de la aplicación.
- **`h1, h2`** → Estilos aplicados a todos los títulos principales y secundarios.
- **`font-size`** → Tamaño de la letra (ej: 56px).
- **`margin: 32px 0;`** → Márgenes superior e inferior. El `0` quita margen izquierdo y derecho.
- **`@media (max-width: 1024px)`** → Regla responsive: los estilos dentro se aplican solo en pantallas más pequeñas que 1024px.
- **`.card`** → Clase reutilizable para crear cajas (tarjetas) con fondo, borde, sombra y redondeo.
- **`border-radius`** → Redondea las esquinas de un elemento.
- **`padding`** → Espacio interno dentro de un elemento.
- **`box-shadow`** → Agrega sombra al elemento.
- **`var(--nombre)`** → Forma de usar las variables definidas en `:root`.

### Propósito general del CSS actual:
- Definir colores y fuentes globales con variables.
- Centrar y limitar el ancho del contenido.
- Hacer la página responsive.
- Crear un diseño limpio y moderno con tarjetas.
